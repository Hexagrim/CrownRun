# CrownRun

A 2d local multiplayer game where you steal the crown and run around as a 2d physics based active ragdoll.
The game was made in 26 hrs.
 \
  \
Basic gameplay is just running around, stealing the crown by touching the king and whoever has the crown at the end of a 2 minute round wins the round!
___

## Controls

RED: Arrow Keys \
BLUE: W AD \
YELLOW: T FH \
GREEN: I JL 

___

## Maps

There are 2 maps:

Red Valley, \
Blue Cave 
___
## Players

The game can be played by 2 , 3 or 4 players using a singular keyboard.

___

## Powerups

There are 3 main powerups:

1. SpeedBoost \
2. JumpBoost \
3. Sheild

The Speedboost makes you fassssst

The JumpBoost nearly triples your jump height 

The Shield protects your crown from being stolen.
